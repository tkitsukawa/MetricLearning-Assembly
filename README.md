## システム概要
人手による製品の組み立て作業において、カメラ情報から作業進捗を推定するシステム。
<br>
組立作業者を見て進捗を推定するのではなく、製品状態を基に進捗ステップを推定する。
<br>
進捗推定は少ない学習データでも特徴を抽出できるようにするためDeep Metric Learningを利用する。
